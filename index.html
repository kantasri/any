Getting Started with Python for Data Analysis

Published on October 27, 2023 | Author: Dr. A. Researcher | Topic: Programming, Data Analysis

This guide provides a quick start for TU Delft researchers who are new to using Python for their data analysis workflows. We'll cover the essential setup and first steps to empower your research.

1. Setting Up Your Environment

We recommend using Conda for package management as it simplifies installing scientific libraries. The TU Delft provides access to many software tools; check the IT Guide for more information.

Create a new Conda environment with the necessary packages:

conda create -n my-data-analysis python=3.9 pandas numpy matplotlib jupyterlab
conda activate my-data-analysis

2. Your First Analysis with Pandas

Pandas is the cornerstone library for data manipulation in Python. Let's see how to load a simple CSV file.

import pandas as pd

# Load data from a CSV file
# Pro-Tip: Always inspect your data after loading!
data = pd.read_csv('experiment_results.csv')
print(data.head()) # Displays the first 5 rows

The output might look something like this:

Sample Temperature (K) Conductivity
A12935.2
A22955.5
